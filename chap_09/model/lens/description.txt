1. R.lens 함수로 객체의 특정 속성에 대한 렌즈를 만든다.
2. 랜즈를 R.view 함수에 적용해 속성값을 얻는다
3. 렌즈를 R.set 함수에 적용해 속성값이 바뀐 새로운 객체를 얻는다
4. 렌즈와 속성값을 바꾸는 함수를 R.over 함수에 적용해 값이 바뀐 새로운 객체를 얻는다.


lens라는건 예전 언어에서 쓰던 기술중 하나
TS에서 lens를 만들어서 부가적인 기능들을 사용할 수 있음
그 기능중 대표기능이 getter/setter

getter는 lens로 만든 값에 R.view를 적용해서 사용함
setter는 lens로 만든 값에 R.set을 적용해서 사용함 
over는 lens와 func을 받아서 새로운 객체를 리턴하는 기능임
set은 value를 받는데 over는 function을 적용해서 새로운 객체를 리턴하는 차이점이 있는 듯 함
